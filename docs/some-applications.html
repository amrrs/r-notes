8.5 Some Applications

The R-package tigerData (HomerWhite 2017) contains the data set reviews, a collection of user-reviews on Amazon for seven bestsellers. Learn more about the data set as follows:

library(tigerData)
help(reviews)

References

HomerWhite. 2017. TigerData: GC Statistics Datasets. https://github.com/homerhanumat/tigerData.
